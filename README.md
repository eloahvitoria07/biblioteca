# biblioteca
Este repositório, chamado “Biblioteca”, reúne o desenvolvimento de um site criado para receber e organizar informações sobre leituras de obras diversas feitas pelo usuário,
funcionando como um acervo digital que facilita a visualização do histórico de leituras.
O projeto tem como objetivo oferecer uma forma simples de registrar cada obra, permitindo adicionar dados como título, autor, gênero, data de leitura e observações pessoais.
